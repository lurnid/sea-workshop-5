About Project Manager

This is a simple project management application built with Flask as part of the Software Engineering & Agile module.

It demonstrates the Model-View-Controller (MVC) pattern and basic database operations using Flask-SQLAlchemy.

Features

  • View all projects
  • Add new projects
  • Edit existing projects
  • Delete projects
  • Organise projects by category
